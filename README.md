# IDSK 3.0 React Component Library

## Available Scripts

### Install dependencies

```
npm install
```

### Format code with Prettier

```
npm run prettier
```

### Run linter

```
npm run lint
```

### Run unit tests

```
npm run test
```

### Run storybook locally

```
npm run storybook
```

### Build storybook

```
npm run build-storybook
```

### Build the library

```
npm run rollup
```

### Publish the library

```
npm publish
```
